# 💳 GenAI Fraud Analyst

An Autonomous AI-powered financial fraud detection system built using a multi-agent architecture.

## 🚨 Problem
Traditional fraud detection systems lack explainability, compliance validation, and automated action.

## 💡 Solution
We built a multi-agent system that:
- Detects fraud using transaction patterns
- Checks compliance violations
- Explains decisions in human-readable form
- Logs every action for auditability
- Takes automated actions (block/review/approve)

## 🧠 Architecture

Agents:
- Fraud Detection Agent
- Compliance Agent
- Explanation Agent
- Audit Agent
- Action Agent

## ⚙️ Tech Stack
- Python
- Streamlit

pip install -r requirements.txt
